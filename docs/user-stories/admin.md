# Admin User Stories

## Role

The admin is an internal Gridline user responsible for configuration,
operational oversight, and permission auditing.

## Stories

### Inspect Role Assignments

As an admin, I can inspect user role assignments so permissions can be audited.

Status: Proposed

### Review Permission Matrix

As an admin, I can review the role-to-permission matrix so I can understand what
each role is allowed to do.

Status: Proposed

### Manage Baseline Configuration

As an admin, I can review customers, sites, providers, users, and role
assignments so the operational baseline can be maintained.

Status: Deferred
