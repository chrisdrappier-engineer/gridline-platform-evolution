# Company Management User Stories

## Role

Company management represents internal Gridline leaders responsible for
operational performance, provider quality, customer health, and cost visibility.
These stories may initially be implemented under admin access and later split
into a dedicated management role if the product needs that distinction.

## Stories

### View Cost Dashboard

As company management, I can view total service cost by customer, site,
provider, cost type, and date range so I can understand where operational spend
is concentrated.

Status: Proposed

### View Provider Response Time

As company management, I can view average and percentile provider response times
so I can identify providers that respond quickly or create operational risk.

Status: Proposed

### View Provider Resolution Percentage

As company management, I can view provider resolution percentage so I can
compare provider effectiveness and identify repeat follow-up patterns.

Status: Proposed

### View Provider Rating Trends

As company management, I can view facility manager ratings and comments by
provider so customer experience can influence provider evaluation.

Status: Proposed

### View SLA Risk Summary

As company management, I can view requests approaching or exceeding response and
resolution targets so operational leaders can intervene before customer impact
grows.

Status: Proposed

### Export Management Report

As company management, I can export cost, response, resolution, and rating
summaries so leadership can review trends outside the application.

Status: Proposed
